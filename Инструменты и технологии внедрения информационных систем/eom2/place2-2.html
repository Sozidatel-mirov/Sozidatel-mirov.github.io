Теоретический материал:


Элементы интерфейса, их еще называют элементы управления, которые используются при разработке пользовательского интерфейса.


Основые элементы пользовательского интерфейса:


  1. - Главное меню - это элемент управления информационной системы, предназначенный для быстрого доступа к основным функциям и элементам окон программы.
  2. - Окно авторизации - это элемент процесса проверки прав пользователя на осуществление определенных действий
  3. -Окно обратной связи - это диалоговое окно для отправки ответной реакции пользователей
  4. - Диалоговые окна - это элемент пользовательского интерфейса, тип окна, который используется для обеспечения связи и взаимодействия между пользователем и программным обеспечением
  5. - Система навигации - это часть пользовательского интерфейса, элементы , котороц обеспечивают возможность переходов между окнами.
