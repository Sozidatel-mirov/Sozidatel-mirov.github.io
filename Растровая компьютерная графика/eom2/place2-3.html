2. Используя инструмент «Прямолинейное лассо» (L) необходимо вырезать первый пазл. Для этого:

1. Переместите курсор инструмента «Прямолинейное лассо» на холст и щелкните, чтобы задать начальную точку области выделения.
2. Перетащите, чтобы нарисовать прямой отрезок для начала выделения объекта. Щелкните, чтобы задать конечную точку первого отрезка. Продолжайте рисовать следующие отрезки до тех пор, пока объект не будет полностью выделен.
3. Чтобы замкнуть границу выделения, переместите курсор инструмента «Прямолинейное лассо» к начальной точке и щелкните ее.
4. С помощью инструмента «Перемещение» (V) передвиньте пазл на зеленое поле.
