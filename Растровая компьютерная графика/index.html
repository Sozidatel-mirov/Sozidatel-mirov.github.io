Ключевые слова:

Компьютерная графика Занимается созданием изображений и искусства с помощью компьютеров.
Обработка изображения Осуществляться как для получения изображения на выходе, так и для получения другой информации.
Классификация компьютерной графики Отличается принципами формирования изображения при отображении на экране монитора или при печати на бумаге.
Средства графической интерпретации Средства визуального анализа табличной информации в виде диаграмм.
Деловая графика Область компьютерной графики, предназначенная для наглядного представления различных показателей работы учреждений.
Иллюстрация Это изображение, сопровождающее, дополняющее и наглядно разъясняющее текст литературного или научного произведения; разновидность изобразительных искусств.
Чертежи Это документ, содержащий контурное изображение изделия и другие данные, необходимые как для изготовления, контроля и идентификации изделия, так и для операций с самим документом.

Рекомендации для обучающегося (самостоятельная работа):

Повторите ранее пройденный материал.
Ознакомьтесь с теоретическим материалом по новой теме. Просмотрите видеолекцию по теме «Работа с растровойграффикой»
Если в процессе изучения материала возникнут вопросы, запишите их для дальнейшего обсуждения с преподавателем и/или с группой обучающихся.
Обратите внимание на основные вопросы, подтверждающие взаимосвязь знаний и умений при изучении модуля:

  • Обработка изображения
  • Классификация компьютерной графики
  • Средства графической интерпретации
  • Деловая графика
  • Иллюстрация
  • Чертежи

При необходимости прослушайте/просмотрите материал еще раз. Возвращайтесь к наиболее сложным аспектам темы. Соблюдайте здоровьесберегающий режим: чередуйте работу с электронными носителями с отдыхом и гимнастикой для глаз.

Рекомендации для преподавателя:

Преподаватель должен

  • организовать просмотр видеолекции и последующую беседу по уточнению и конкретизации первичных знаний;
  • создать условия для освоения обучающимися знаний в форме интерактивной деятельности;
  • визуализировать подачу учебного материала с помощью иллюстраций, анимаций.
  • формировать у обучающихся мотивацию к усвоению нового материала, образованию как основному инструменту достижения личного и профессионального успеха;
  • осуществить постановку учебной проблемы: решение типичных проблем в построении растрового изображения;
  • организовать анализ изложенной учебной задачи по определению проблем построения растровых объектов;
  • познакомить обучающихся с методами определения необходимости применения растровых объектов;
  • показать взаимосвязь растровых объектов, значение, преимущества, недостатки их применения;
  • раскрыть роль, значение растровой графики;
  • продемонстрировать применение растровой графики на конкретном примере;
