Развертывание Active Directory

Шаг 3

4/5

Развертывание контроллера домена на DC1



  1. В оснастке server manager переходим на вкладку manage и выбираем add roles and Features
  2. Выбираем тип установки — Role-Based on feature-based installation
  3. Выбираем на какой сервер ставим роль — DC1
  4. Выбираем роль - Active Directory Domain Services
  5. В появившемся окне нажимаем Add Features, чтобы установились необходимые дополнения.
  6. Нажимаем Next до окна Confirm installation selections.
  7. Ставим галку Restart the destination server automatically if required
  8. Сохраняем скрипт настроек для отчета
  9. Нажимаем Install


После установки роль настраиваем
