Развертывание Active Directory

Шаг 4

5/5
  1. Для этого нажимаем флажок в верху ServerManager и выбираем Promote this server in domain controller
  2. В окне Deployment Configuration Указываем Add a new forest и имя домена Kazan.wsr
  3. Указываем пароль для восстановления и удаления базы данных ADDS. Можно указать P@ssw0rd.
  4. Соглашаемся с тем, что система установить на DC1 DNS сервер
  5. Указываем NetBIOS имя KAZAN.
  6. Оставляем пути как есть
  7. Сохраняем скрипт настроек для отчета
  8. Ждем окончания проверки конфигурации и нажимаем Install
  9. После установки и перезагрузки мы сможем входить в систему как пользователь локального компьютера и пользователь домена
